# ARF API Control Plane Documentation

Welcome to the ARF API! This API exposes the **Agentic Reliability Framework (ARF)** reliability engine as a deployable service. It allows you to report incidents, simulate healing intents, query system risk, and retrieve historical data.

## Quick Start

1. **Install the API** (see [development guide](development.md)) or use the hosted version.
2. **Authenticate** using an API key in the `X-API-Key` header ([details](authentication.md)).
3. **Explore endpoints** in the [endpoints reference](endpoints.md).
4. **See examples** in [examples.md](examples.md).

The interactive OpenAPI documentation is available at `/docs` when the server is running.
